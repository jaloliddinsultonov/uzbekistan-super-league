# 🏆 Uzbekistan Super League

A mobile football management simulation game built with Unity,
featuring all 16 clubs from the Uzbekistan Super League.

## 🎮 Play Now
[Play on itch.io] https://aladdin-jakubowski.itch.io/uzbekistan-super-league

## 📱 About
Lead any USL club through a full 30-matchday season.
Watch live match simulations, track the league table,
and compete to become champion!

## ⚽ Features
- 16 real USL clubs with authentic logos
- Full round-robin season (30 matchdays)
- Live match simulation with real-time goal feed
- Dynamic league table with full stats
- Unique Attack, Defense & Morale system per club
- Background music & goal sound effects
- Audio settings

## 🛠️ Built With
- Unity 6
- C#
- TextMeshPro
